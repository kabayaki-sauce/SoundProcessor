# CODE OF CONDUCT

## 行動規範の目的
本行動規範は、誰もが安全かつ建設的に参加できるコラボレーション環境を維持することを目的とします。

## 私たちの約束
私たちは、年齢、体格、障がい、民族、性別、性自認、性的指向、経験レベル、教育、社会経済的地位、国籍、個人的外見、信条、宗教に関わらず、
すべての参加者にとってハラスメントのない環境を提供します。

## 望ましい行動

- 異なる意見や経験への敬意を示す
- 建設的なフィードバックを行う
- ミスを認め、学習と改善を優先する
- コミュニティ全体にとって最善の判断を行う

## 許容されない行動

- 侮辱・中傷・差別的発言
- 個人情報の無断公開
- 継続的な攻撃的言動、荒らし行為
- 性的な言動や不適切な接触
- その他、職業的な協業環境として不適切な行為

## 適用範囲

本規範は以下に適用されます。

- 本リポジトリの Issue、PR、Discussion 相当の公開コミュニケーション
- プロジェクト関連のレビュー、コメント、提案
- コミュニティへの公式な代理発言

## 通報方法

違反行為を確認した場合は GitHub Issues で報告してください。

- タイトル例: `COC: 行動規範違反の報告`
- 可能な範囲で事実（発生日時、場所、該当URL、状況）を記載してください。

## 執行ポリシー

管理者は、報告内容に対して次のいずれかの対応を行います。

1. 是正依頼（注意）
2. 警告
3. 一時的な参加停止
4. 永続的な参加停止

対応レベルは、違反の重大性・継続性・再発性を考慮して判断します。
