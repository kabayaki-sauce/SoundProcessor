# SUPPORT

## サポート窓口

問い合わせは GitHub Issues に一本化しています。

## カテゴリとタイトル例

- 利用方法の質問: `Question: ...`
- 不具合報告: `BUG: ...`
- 機能提案: `Feature: ...`
- ドキュメント修正提案: `Doc: ...`
- セキュリティ関連: `Security: ...`
- 行動規範違反の報告: `COC: ...`

## 問い合わせ時の推奨テンプレート

```
### 目的
### 実施内容
### 期待結果
### 実際の結果
### 実行コマンド
### 環境情報（OS、.NET SDK、音声処理ツールのバージョン）
```

## サポート対象外

- 外部ツールの個別ビルドや配布バイナリの再現不能問題
- ローカル環境固有の制約に起因する不具合
- 依存ライブラリ側の仕様変更に関する即時対応保証
